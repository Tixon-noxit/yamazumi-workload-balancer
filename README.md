
#  Yamazumi Board – Полнофункциональная система балансировки нагрузки (секунды)

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/HTML)
[![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> **Yamazumi board (Ямадзуми)** — профессиональный инструмент для визуализации и оптимизации загрузки рабочих постов, операторов или станков. Работает с секундами, поддерживает drag-and-drop, автосохранение и экспорт в Excel.

---

##  Для чего это?

Инструмент предназначен для **Lean-специалистов**, **мастеров участков**, **инженеров по организации труда** и всех, кто занимается:

- ✅ **Балансировкой производственных линий** (сборочные, упаковочные, логистические)
- ✅ **Анализом загрузки персонала** (time studies, хронометраж)
- ✅ **Визуализацией узких мест** (перегруженные посты сразу выделяются цветом)
- ✅ **Быстрым экспериментированием** (перетаскивание операций между постами в реальном времени)

##  Демонстрация (как это работает)

> **Сценарий:** На сборочной линии 6 операторов. Вы засекаете каждую операцию: сварка 45 сек, зачистка 30 сек, контроль 20 сек. Добавляете их на посты. Видите, что Пост 3 перегружен (красный). Просто **перетаскиваете** операцию мышкой на свободный Пост 5 — загрузка выравнивается.

### Интерфейс включает:

- **Левая панель** — форма добавления/редактирования операций + кнопки управления
- **Правая область** — сетка карточек постов (прогресс-бар, суммарное время, список операций)
- **Внизу** — столбчатая диаграмма загрузки всех постов
- **Верхняя панель** — кнопки Excel, Печать, Сброс

- **HTML5 / CSS3** — адаптивная сетка, промышленный стиль, print-стили

##  Как пользоваться (подробно)

### 1. Установите лимит смены
Поле *«Максимальное время поста 100% (секунды)»*.  
По умолчанию **28 800 сек = 8 часов**.  
*Пример:* если смена 7 часов → введите `25200`.

### 2. Добавьте операцию
- **Название** (например, *Сварка рамки*)
- **Время в секундах** (целое число, ≥1)
- **Пост** (выберите из списка Пост 1 … Пост 10)
- Нажмите **«ДОБАВИТЬ / СОХРАНИТЬ»**

### 3. Редактирование операции
- Нажмите **✏️** у любой операции
- Поля формы заполнятся её данными
- Измените название, время или выберите другой пост
- Нажмите ту же кнопку — изменения применятся

### 4. Удаление операции
- Нажмите **🗑️** у ненужной операции
- Подтвердите удаление

### 5. Drag-and-drop (перетаскивание)
- **Зажмите левую кнопку мыши** на любой операции
- **Перетащите** её на карточку другого поста
- Отпустите — операция переместится, загрузка пересчитается автоматически

### 6. Анализ загрузки
- **Прогресс-бар** и процент на каждом посте
- **Столбчатая диаграмма** внизу показывает загрузку всех постов
- Цвета: зелёный (<90%), жёлтый (90–99%), красный (≥100%)

### 7. Экспорт в Excel
- Нажмите **«EXCEL»** на верхней панели
- Скачается файл с детальным отчётом: операции, суммарное время, лимиты, проценты

### 8. Печать
- Нажмите **«ПЕЧАТЬ»**
- Откроется окно предварительного просмотра с оптимизированным для печати видом (без панели управления)

### 9. Сохранение
- Все изменения **автоматически сохраняются** в браузере (localStorage)
- После обновления страницы или перезапуска браузера данные останутся

### 10. Сброс
- Нажмите **«СБРОС»** → подтвердите → все операции будут удалены

##  Настройка под ваш проект

### Изменить количество постов
В коде найдите:
```javascript
const POSTS_COUNT = 10;
```
Измените на нужное значение (например, `6` или `15`).

### Изменить лимит времени по умолчанию
В HTML у поля `id="maxTime"` атрибут `value="28800"`.  
Замените на любое другое число секунд.

### Очистить сохранённые данные в браузере
Если нужно начать с нуля:
- Нажмите кнопку **«СБРОС»** в интерфейсе
- Или очистите localStorage вручную: `localStorage.removeItem("yamazumi_seconds_data")`
